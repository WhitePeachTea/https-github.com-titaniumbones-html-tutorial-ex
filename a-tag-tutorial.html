The simplest solution to add a youtube video

Most of the video you need could be find in Youtube Channel, and they provided a simplest way to embed to your website by using iframe tag.

Right cilck to select the "copy the embed code" option.

You get the code like this <iframe width="854" height="480" src="https://www.youtube.com/embed/LSxElWwWVFE" frameborder="0" allowfullscreen> </iframe>

Paste the code in your HTML and here you go
